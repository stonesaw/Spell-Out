# j2 夏休み課題
属性(Spell)の要素を加えたシューティングゲーム  
属性の相性によってダメージが変わるよ  


## 基本操作
エスケープキー : ゲームの終了

- プレイ中
    | キー | 操作 |
    | --- | --- |
    | W・A・S・D | 移動 |
    | スペース | 攻撃 |
    | スペース(長押し) | オート攻撃 |
    | J・K | 属性の変更 |
    
- メニュー
    | キー | 操作 |
    | --- | --- |
    | 矢印キー | 移動 |
    | エンター | 決定 |
